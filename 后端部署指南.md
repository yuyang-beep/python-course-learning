# 后端服务器部署指南

## 📋 概述

本项目提供了一个 Node.js + Express + Socket.io 的后端服务器，可以完全替代 Firebase。

**优势**：
- 自己控制服务器
- 可以自定义功能
- 更好的性能控制
- 可以部署到自己的服务器上

**需要的技能**：
- 基本的 Node.js 知识
- 命令行操作

---

## 第一步：安装 Node.js

### 如果你还没安装 Node.js

1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载 LTS 版本（Long Term Support）
3. 双击安装程序，按照提示安装
4. 安装完成后，打开命令行（cmd 或 PowerShell）
5. 输入以下命令验证安装：
   ```bash
   node --version
   npm --version
   ```
   如果显示版本号，说明安装成功

---

## 第二步：安装依赖

### 2.1 打开命令行
- **Windows**: 按 Win+R → 输入 cmd → 回车

## 🚀 生产部署

### 部署到云服务器

#### 选项 1：Heroku（最简单）

1. **创建 Heroku 账号**
   - 访问 [Heroku.com](https://www.heroku.com)
   - 注册账号

2. **安装 Heroku CLI**
   - 下载并安装 [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)

3. **登录并创建应用**
   ```bash
   heroku login
   heroku create your-app-name
   ```

4. **部署**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git push heroku main
   ```

5. **查看日志**
   ```bash
   heroku logs --tail
   ```

#### 选项 2：阿里云/腾讯云/AWS

1. **购买云服务器 (ECS/CVM/EC2)**
2. **连接到服务器**
3. **安装 Node.js**
4. **上传项目文件**
5. **运行 npm install && npm start**
6. **配置反向代理 (Nginx)**
7. **配置 SSL 证书**

#### 选项 3：Docker 部署

创建 Dockerfile：
```dockerfile
FROM node:16

WORKDIR /app

COPY package*.json ./
RUN npm install

COPY . .

EXPOSE 3000

CMD ["npm", "start"]
```

构建并运行：
```bash
docker build -t emperor-game-server .
docker run -p 3000:3000 emperor-game-server
```

## ⚠️ 常见问题

### Q: 端口 3000 已被占用
**解决**:
```bash
# 使用其他端口
PORT=8000 npm start
```

### Q: npm install 速度很慢
**解决**:
```bash
# 使用国内镜像
npm install -g cnpm
cnpm install
```

### Q: 服务器无法从外部访问
**解决**:
- 检查防火墙设置
- 检查云服务器安全组规则
- 确保端口已开放

### Q: WebSocket 连接失败
**解决**:
- 检查服务器是否运行
- 检查 socket.io 是否正确初始化
- 查看浏览器控制台错误

### Q: 内存占用越来越大
**原因**: 房间数据没有清理  
**解决**: 实现房间过期清理机制
```javascript
// 定期清理闲置房间
setInterval(() => {
    const now = Date.now();
    rooms.forEach((room, code) => {
        if (now - room.createdAt > 24*60*60*1000) { // 24小时
            deleteRoom(code);
        }
    });
}, 60*60*1000); // 每小时检查一次
```

## 🔌 Socket.io 事件列表

### 客户端发送的事件

| 事件 | 参数 | 说明 |
|------|------|------|
| `join_room` | {roomCode, playerId, playerName} | 加入房间 |
| `create_room` | {} | 创建房间 |
| `assign_role` | {roomCode, playerId, role} | 分配身份 |
| `start_game` | {roomCode} | 开始游戏 |
| `start_round` | {roomCode, round, config} | 开启轮数 |
| `submit_vote` | {roomCode, playerId, round, vote} | 提交投票 |
| `end_round` | {roomCode, round} | 结束轮数 |
| `exit_room` | {roomCode, playerId} | 退出房间 |
| `delete_room` | {roomCode} | 删除房间 |

### 服务器发送的事件

| 事件 | 数据 | 说明 |
|------|------|------|
| `player_joined` | {players} | 玩家加入 |
| `role_assigned` | {playerId, role, assignments} | 身份已分配 |
| `game_started` | {status, currentRound} | 游戏已开始 |
| `round_started` | {status, round, config} | 轮数已开启 |
| `vote_stats_updated` | {round, stats, totalPlayers} | 投票统计更新 |
| `round_ended` | {round, results, players} | 轮数已结束 |
| `player_left` | {playerId, players} | 玩家已退出 |
| `room_deleted` | {} | 房间已删除 |
| `error` | {message} | 错误信息 |
