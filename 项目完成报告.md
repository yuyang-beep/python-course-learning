## 🚀 快速开始指南

### 方案 A：Firebase（推荐新手）

1. **配置 Firebase** (10分钟)
   - 访问 Firebase Console
   - 创建项目和数据库
   - 复制配置

2. **填入配置** (2分钟)
   - 打开 game-admin.html 和 game-player.html
   - 粘贴 Firebase 配置

3. **开始游戏** (1分钟)
   - 打开两个浏览器标签页
   - 管理者创建房间
   - 玩家加入并开始游戏

**总耗时**: 13分钟

### 方案 B：自建服务器（推荐正式应用）

1. **安装依赖** (2分钟)
   ```bash
   npm install
   ```

2. **启动服务器** (1分钟)
   ```bash
   npm start
   ```

3. **打开客户端** (1分钟)
   - 打开 game-admin-server.html
   - 打开 game-player-server.html

**总耗时**: 4分钟
