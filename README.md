## 运行方式

每个文件都是独立脚本，直接用 Python 运行即可，例如：

```bash
python Day09/list2.py
```

需要 Python 3.8 及以上版本；练习中用到的都是标准库（如 `random`），无需额外安装第三方包。

## 说明

代码里带有较多中文注释，记录了每个知识点的语法说明和示例输出，方便日后复习。部分文件末尾附带小练习或实战小项目（例如 Day09 的双色球随机选号）。
